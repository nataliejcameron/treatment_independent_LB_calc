# treatment_independent_LB_calc
This repository contains the source code for a R Shiny calculator for predicting chance of pregnancy leading to live birth before starting treatment for couples with infertility.
> **Note:** The original patient dataset is not contained in, or required for, this calculator. It is built using the final model formula.

## Live calculator

The interactive calculator is hosted on shinyapps.io and is available here:
**[https://njcameron.shinyapps.io/riskcalc_app/](https://njcameron.shinyapps.io/riskcalc_app/)**

---

## Publication

This tool was developed as part of the following publication:
* **Title:** "Predicting the chance of treatment-independent live birth in couples diagnosed with different types of infertility"
* **Submitted to:** Human Reproduction Open, in review.
* **Authors:** N.J. Cameron, D.J. McLernon, S. Bhattacharya.

## Run this app locally:

If you wish to run the app on your own computer, simply clone this repository, open RStudio in the project folder and run the app (e.g. `shiny::runApp()`).
